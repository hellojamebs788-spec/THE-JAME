# 💰 Money Tracker

เว็บรายรับรายจ่าย พร้อมระบบอ่านสลิป OCR

## ฟีเจอร์
- เพิ่มรายรับ / รายจ่าย / หนี้สิน
- สรุปเงินคงเหลือ
- เก็บข้อมูลใน LocalStorage
- อ่านสลิปด้วย OCR
- ใช้งานบน GitHub Pages ได้

---

## วิธีใช้งาน

### 1. Upload ขึ้น GitHub

สร้าง Repository ใหม่ เช่น:

money-tracker

อัปโหลดไฟล์ทั้งหมดขึ้น GitHub

---

### 2. เปิด GitHub Pages

ไปที่:

Settings → Pages

เลือก:

Deploy from branch

เลือก branch:

main

เลือก folder:

/root

Save
